**Inhoud:**
- **Back-end (Node.js)**:
  - BeerMicroservice
  - OrderMicroservice
  - Gateway

- **Front-end**
  - React front-end
---

**Installatie**

1. clone de repository
2. Installeer dependencies voor elk project d.m.v. **npm install**
3. Start de BeerMicroservice d.m.v. **node index.js**
4. Start de OrderMicroservice d.m.v. **node index.js**
5. Start de Gateway d.m.v. **node index.js**
6. Start de front-end d.m.v. **npm start**
